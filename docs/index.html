Yirgacheffe

Like Java but for pretentious people.

Build

mvn package produces the executable executable/target/yirgacheffe.

Run

Compile classes:

yirgacheffe MyClass.yg AnotherClass.yg

Execute:

yirgacheffe --run MyClass

Repl:

yirgacheffe --repl
